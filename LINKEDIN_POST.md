# 🎯 TalentMatch: Open-Source AI Hiring Tool

## 📍 Post Caption

---

I just shipped **TalentMatch** — an enterprise-grade AI hiring tool that's 100% open-source and runs entirely on your machine. 🚀

### The Problem 🤔
Recruiting tech talent is broken:
- Cloud-based tools cost **$100s monthly** in LLM API fees
- Your candidates' resumes sit in third-party servers (privacy nightmare)
- Generic ATS matching misses great fits
- You're locked into expensive vendor platforms
- AI bias in hiring isn't addressed — it's hidden

### The Solution ✨
**TalentMatch** flips the script:

✅ **Local-First Architecture** — Everything runs on your machine. Zero cloud calls. Zero API bills.
✅ **Dual-Stage Scoring** — Semantic embeddings (40%) + structured LLM analysis (60%) = precision hiring
✅ **Multi-JD Support** — Screen candidates against 1 or 5 job descriptions simultaneously
✅ **Bias Reduction** — Resume anonymization removes names/emails/phone before scoring
✅ **Deep Analytics** — Radar charts, keyword gap analysis, candidate comparisons
✅ **Full Transparency** — Complete audit trails of every LLM decision
✅ **Privacy-First** — Your data never leaves your machine. No accounts. No sign-ups.

### How It Works ⚙️

1. **Upload** resumes (PDF/DOCX) + job descriptions
2. **Configure** your local LLM (via LM Studio on port 1234)
3. **Run** screening — watch real-time progress
4. **Analyze** results — radar charts, shortlist candidates, export CSV
5. **Save** sessions locally — restore anytime

- **Streamlit** (beautiful UI)

### Key Features 🌟

**For Recruiters:**
- Batch screen 50+ candidates in one session
- Per-dimension scoring (Skills, Experience, Education, Culture Fit)
- Candidate comparison (2-3 way side-by-side)
- Shortlist markers + internal notes
- CSV export for ATS integration
- LLM call audit log for compliance

**For Job Seekers:**
- Generate ATS-friendly resumes
- Analyze job market requirements
- Self-screen against postings
- Track resume versions locally
- Identify skill gaps before applying
